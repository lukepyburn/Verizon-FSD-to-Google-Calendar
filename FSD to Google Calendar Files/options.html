Verizon FSD to Google Calendar Options

Select the FSD details that will appear in Google Calendar Event Title:

Current Date
Customer Name
Phone Number
Job To Do

Select date and time options:

Set as current date and time.
Set as date and time from the FSD entry.
Set as an all-day event on the current date.
Set as an all-day event on the date from the FSD entry.

**Default event length will be 1 hour unless one of the all-day options has been selected.**

Auto copy job description.
